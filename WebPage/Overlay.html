First Start

This is the first start of the HomeLoader Overlay.

The Overlay needs HomeLoader to be already installed to work properly.

HomeLoader needs to be started at least one time after installation and an new Home App needs to be set through it. This can be also the original SteamVR Home, it just should not be set to 'default' to work properly.
